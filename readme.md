# Web Screenshot Comparison
====

urls.csvに記載されたA/B列のURLについて
WebDriverを使用してスクリーンショットを撮影し、比較します。
差分は赤い枠で囲まれて表示されます。


## Usage
pip install selenium
pip install webdriver-manager
pip install numpy
pip install Pillow

>python compare_websites.py
